# Snake_Water_Gun_Project 

A simple Python command-line game using the `random` module.

##  How to Play
1. Run the script: `python main.py`
2. Enter your choice:
   - **S** for Snake
   - **W** for Water
   - **G** for Gun

##  Rules
- **Snake** beats Water
- **Water** beats Gun
- **Gun** beats Snake

##  Features
- Built with Python.
- Uses Dictionary mapping for game logic.
- Random AI opponent.
